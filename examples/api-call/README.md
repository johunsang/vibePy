# VibeLang API Call Examples

Run any example:
```bash
python3 -m vibelang run <file>
```

Examples
- `get_json.vbl.json` GET and parse JSON
- `get_query_params.vbl.json` GET with query params
- `post_json.vbl.json` POST JSON body
- `bearer_auth.vbl.json` Bearer token auth
- `custom_headers.vbl.json` Custom headers
- `timeout_retry.vbl.json` Retry + timeout
